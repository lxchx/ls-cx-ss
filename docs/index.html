ls-cx-ss

A single-file Codex session helper with GitHub Pages distribution. You can run it straight from a URL via python3, or install one command locally.

Use Directly From URL

curl -fsSL https://lxchx.github.io/ls-cx-ss/ls-cx-ss.py | python3 -
curl -fsSL https://lxchx.github.io/ls-cx-ss/ls-cx-ss.py | python3 - list
curl -fsSL https://lxchx.github.io/ls-cx-ss/ls-cx-ss.py | python3 - tui
curl -fsSL https://lxchx.github.io/ls-cx-ss/ls-cx-ss.py | python3 - list --all-cwds
curl -fsSL https://lxchx.github.io/ls-cx-ss/ls-cx-ss.py | python3 - list --search manga

With no subcommand, ls-cx-ss now defaults to the TUI. Use -h for help.

Install From URL

curl -fsSL https://lxchx.github.io/ls-cx-ss/install.py | python3 -
~/.local/bin/ls-cx-ss list
